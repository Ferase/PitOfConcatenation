for (Iteration of Objects)

Objects are a bit different than arrays.

The same for loop as before just doesn’t work with objects:

for(var i = 0; i < 1000; i++) { }

Objects aren’t a true sequential collection and so for looping over them is a bit different. See the Looping documentation for more info.

You need to first determine the object’s length property and get the values of each property. This is needed to loop over the object properties. You also need to loop over a variable that will keep track of which properties you are currently iterating over.

The final step is to iterate over the object using the current values of the property. You need to stop when the variable reaches the end of the properties.

There are also things like ES6’s map, filter, and reduce functions, but I won’t get into them. See the Array documentation for more info.
